# Epstein Files Database 🗂️

A SQL Server database practice project modeling public figure 
connections, events, locations, and flight logs based on 
publicly available court documents and records.

## Purpose
This project was built to practice SQL database design, 
normalization, and complex query writing including JOINs, 
aggregations, and subqueries.

## Tables
- **People** — Public figures and their details
- **Locations** — Properties and locations involved
- **Connections** — Relationships between individuals
- **Events** — Documented gatherings and meetings
- **EventParticipants** — Who attended each event
- **FlightLogs** — Aircraft passenger records

## Technologies Used
- SQL Server 2022
- SQL Server Management Studio (SSMS)
- GitHub for version control

## Skills Demonstrated
- Database schema design
- Primary and foreign key relationships
- Complex JOIN queries
- Aggregation and grouping
- Subqueries
- Data normalization
